# Commit Rules

## Types

| Type | Meaning |
|---|---|
| feat | 新功能 |
| fix | 修复问题 |
| refactor | 重构 |
| docs | 文档修改 |
| test | 测试相关 |
| style | 格式修改 |
| chore | 杂项 |

## Good Examples

```text
fix(import): handle null filename during upload
```

```text
feat(export): support async staff export tasks
```
